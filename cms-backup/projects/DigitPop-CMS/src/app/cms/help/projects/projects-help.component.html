Projects

A project is used to generate a shoppable video and the projects page lists all of your projects with analytics. All fields are sortable and can be searched by entering text in the filter. Click on the + button to start a new project in the project wizard. Click on the edit icon in any of the project rows to edit the project in the wizard.

Project Analytics

Each project displays analytics of marketing insights. Click on a project to expand the analytics to a product group level. Each product group is also expandable to show product level analytics.

Project Analytics - Products

The product analytics show how many times a product has been selected and also how many times the 'Buy Now' button has been pressed to navigate to the shopping cart.
